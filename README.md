# TP ERP — Comparaison de SAP S/4HANA et Microsoft Dynamics 365

## Présentation
Ce dépôt contient un TP réalisé dans le cadre du cours de Progiciel de Gestion Intégrée (ERP) du Master 1 Management des Systèmes d’Information à Grenoble IAE.

L’objectif de ce travail était de comparer deux ERP du marché, SAP S/4HANA et Microsoft Dynamics 365, afin d’analyser leur positionnement, leurs caractéristiques fonctionnelles, leur architecture technique, ainsi que leur expérience utilisateur.

## Contexte du projet
Le projet s’inscrit dans une mission de conseil fictive pour la holding Hold&Win, un groupe multisectoriel en transformation digitale composé de plusieurs entreprises de tailles différentes : TPE, PME, ETI et grande entreprise.

Dans ce contexte, l’enjeu était d’identifier la ou les solutions ERP les plus adaptées selon la taille, la maturité et les besoins métier de chaque structure.

## ERP analysés
Les deux solutions étudiées sont :
- **SAP S/4HANA**
- **Microsoft Dynamics 365**

Ce choix permet de comparer :
- un ERP très robuste, adapté aux environnements complexes et fortement intégrés ;
- une solution plus modulaire, flexible et accessible, particulièrement pertinente pour des structures intermédiaires.

## Périmètre fonctionnel étudié
L’analyse comparative porte sur deux modules clés :
- **Ventes / Distribution**
- **Production & Logistique**

## Axes d’analyse
L’étude a été menée selon plusieurs dimensions :

### 1. Présentation générale des solutions
- positionnement de l’éditeur
- entreprises cibles
- modes de déploiement
- logique globale de la solution

### 2. Comparaison fonctionnelle
- couverture des processus métiers
- différences entre les workflows proposés
- adaptation selon la taille de l’entreprise

### 3. Architecture technique
- cloud / on-premise / hybride
- technologies et bases de données
- intégration avec d’autres outils
- interface utilisateur

### 4. Expérience utilisateur (UX/UI)
- facilité de prise en main
- fluidité de navigation
- personnalisation
- accessibilité mobile
